Project 180 Home HIIT Website

How to use:
1. Open index.html in Chrome, Safari, or Edge.
2. Use Dashboard, Workouts, Timer, Log, and Progress tabs.
3. Workout logs are saved locally in your browser using localStorage.
4. Use the Export JSON button to back up your logs.

To make this a real website:
- Upload this folder to Netlify, Vercel, GitHub Pages, or your own hosting.
